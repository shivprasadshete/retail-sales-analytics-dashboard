# Retail Sales Analytics Dashboard

## 📊 Project Overview
This project is an end-to-end Retail Sales Analytics solution built using Python, MySQL, and Power BI.

The objective was to analyze sales performance, growth trends, and customer behavior to generate business insights.

---

## 🔧 Tech Stack
- Python (Pandas, NumPy)
- MySQL
- SQLAlchemy
- Power BI
- DAX

---

## 📈 Dashboard Pages

### 1️⃣ Executive Sales Overview
- Total Revenue, Profit, Orders, Profit Margin
- Monthly Sales Trend
- Revenue vs Profit Matrix
- Category Contribution

### 2️⃣ Revenue & Growth Analysis
- Month-over-Month Growth
- Revenue Comparison
- Top Products Analysis

### 3️⃣ Customer Behaviour Insights
- Revenue by Customer City
- Top Revenue Generating Customers
- Purchase Frequency Distribution

### 4️⃣ RFM Customer Segmentation
- RFM Scoring Model
- Customer Segmentation (Champions, Loyal, At Risk, Low Value)
- Revenue Contribution by Segment

---

## 🧠 Key Insights
- Identified high-value and at-risk customers
- Analyzed profitability trends across categories
- Implemented time-intelligence functions using DAX
- Built interactive multi-page BI dashboard
